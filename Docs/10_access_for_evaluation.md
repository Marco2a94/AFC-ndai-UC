# Access for evaluation

## Local evaluation (recommended)
Teachers can evaluate locally by cloning the repo and following `03_runbook.md`.

Minimum requirements:
- Docker Desktop
- Ability to open ports: 3000, 8000, 5432

## What to provide to teachers
- Repository link (Git)
- The `.env` template (do not commit passwords)
- The runbook command sequence
- The Metabase URL and (if applicable) a default admin user for the demo

## If you deploy to a cloud VM
Provide:
- Public URL for Metabase
- Public URL for FastAPI
- Credentials for the evaluator (read-only if possible)
- A short checklist to reproduce the demo (load data, refresh dashboard)
